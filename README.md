# Kingly Drupal

Another Drupal 10 project that uses the Kingly recipe for a starter site.

# Getting started

## Copy local ddev config:

```bash
cp .ddev/example.config.local.yaml .ddev/config.local.yaml
```

## Start the project:

```bash
ddev start
```

## Destroy and rebuild the DDEV environment

```bash
ddev rebuild
```
